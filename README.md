# MertOS Dashboard

MertOS Dashboard, tarayıcı başlangıç sayfası olarak kullanılmak üzere geliştirilmiş kişisel bir masaüstü panelidir.

## Özellikler

- Canlı saat ve tarih
- Günlük odak kartı
- Hızlı kısayollar
- Mini yapılacaklar listesi
- Pomodoro zamanlayıcı
- Rastgele motivasyon cümlesi
- Dark / neon / glassmorphism tasarım
- Responsive arayüz
- LocalStorage ile kalıcı veri saklama

## Teknolojiler

- HTML
- CSS
- JavaScript

## Kullanım

Projeyi lokal olarak çalıştırmak için `index.html` dosyasını tarayıcıda açman yeterlidir.

Live Server kullanıyorsan:

```bash
Open with Live Server
```
